## 1. Data સંગ્રહ

Wolfgang Rush plugin કુટુંબ **તમારો કોઈ data એકત્ર કરતું નથી, સંગ્રહિત કરતું નથી, કે પ્રસારિત કરતું નથી** — પ્રકાશકને કે કોઈ ત્રીજા પક્ષને નહીં.

## 2. Plugins કેવી રીતે કામ કરે છે

દરેક plugin **સંપૂર્ણ રીતે સ્થાનિક** તમારા Claude Code installation માં ચાલે છે. Plugin ના agents, skills, અને templates તમારા સ્થાનિક filesystem માંથી વાંચવામાં આવે છે. તમે પૂરા પાડેલા case-folder contents (facts, prior pleadings, statutory anchors, prayer briefs, exhibits) સ્થાનિક રીતે Claude Code દ્વારા process થાય છે અને સામાન્ય Claude Code operation ના માર્ગે ફક્ત તમારી અને Anthropic ની Claude API વચ્ચે પ્રસારિત થાય છે.

પ્રકાશક આ plugins સંબંધિત **કોઈ** server, telemetry endpoint, analytics service, error-reporting service, license-verification service, કે data-collection infrastructure ચલાવતા નથી.

## 3. Anthropic ને મોકલવામાં આવેલ Data

Plugin ના ઉપયોગ સાથે user prompts અને case-folder contents તમારા પોતાના Claude Code installation દ્વારા Anthropic ની Claude API ને મોકલવામાં આવે છે. તે data flow **Anthropic ની Privacy Policy** હેઠળ સંચાલિત છે, જે https://www.anthropic.com/legal/privacy પર ઉપલબ્ધ છે.

પ્રકાશક તે data flow માં પક્ષકાર નથી, user prompts કે case-folder contents માટે કોઈ ઍક્સેસ નથી, અને કોઈ generated output ની નકલ મેળવતા નથી.

## 4. કોઈ user account નહીં

Plugins માટે કોઈ user account, login, કે ઓળખ પગલા જરૂરી નથી. કોઈ registration, email signup, કે licence-key activation નથી.

## 5. Source code પારદર્શિતા

સંપૂર્ણ plugin source code MIT License હેઠળ https://github.com/Wolfgangrush પર પ્રકાશિત છે. વપરાશકર્તાઓ, system administrators, અને security reviewers કોઈપણ સમયે plugin code ની દરેક પંક્તિ તપાસી શકે છે.

## 6. વ્યવસાયિક ગુપ્તતા

વપરાશકર્તાઓ — ખાસ કરીને વકીલો અને law-firm staff — એ ધ્યાન રાખવું જોઈએ કે Claude Code ને આપેલા case-folder contents માં Bharatiya Sakshya Adhiniyam 2023 ની કલમ 132 / Indian Evidence Act 1872 ની કલમ 126 હેઠળ advocate-client confidentiality કે અન્ય વ્યવસાયિક-ગુપ્તતા જવાબદારીઓ આકર્ષે તેવી સામગ્રી હોઈ શકે છે. Plugin ને કઈ સામગ્રી પૂરી પાડવી તે નિર્ણય તમારી વ્યવસાયિક જવાબદારી છે.
