# Privacy Policy — Wolfgang Rush Plugin Family

**Plugin Family:** Wolfgang Rush — open-source Indian legal-drafting plugins for Claude Code
**Publisher:** Rushikesh R. Mahajan, publishing as Wolfgang Rush
**Publisher contact:** advrushikeshravindramahajan@gmail.com
**Effective date:** 17 May 2026

## 1. Data collection

The Wolfgang Rush plugin family **does not collect, store, or transmit any user data** to the Publisher or to any third party operated by the Publisher.

## 2. How the plugins operate

Each Wolfgang Rush plugin runs **entirely locally** within the user's Claude Code installation. The plugin's agents, skills, statutory references, and pleading templates are read from the user's local filesystem. Case-folder contents supplied by the user (facts, prior pleadings, statutory anchors, prayer briefs, exhibits) are processed locally by Claude Code and transmitted only between the user and Anthropic's Claude API in the normal course of Claude Code operation.

The Publisher does **not** operate any server, telemetry endpoint, analytics service, error-reporting service, license-verification service, or data-collection infrastructure of any kind in connection with these plugins.

## 3. Data sent to Anthropic

Use of the plugins necessarily involves user prompts and case-folder contents being sent to Anthropic's Claude API by the user's own Claude Code installation. That data flow is governed by **Anthropic's Privacy Policy**, available at https://www.anthropic.com/legal/privacy, and by Anthropic's commercial terms applicable to the user's Claude Code subscription.

The Publisher is not a party to that data flow, has no access to user prompts or case-folder contents, and receives no copy of any generated output.

## 4. No user accounts

The plugins do not require, create, or maintain any user account, login, or identification step. There is no registration, no email signup, and no licence-key activation.

## 5. Source code transparency

All plugin source code is published under the MIT License at https://github.com/Wolfgangrush. Users, system administrators, and security reviewers may inspect every line of plugin code at any time. There is no closed component, no obfuscated binary, and no remote-loaded code path.

## 6. Children

The plugins are professional legal-drafting tools intended for use by qualified advocates, law-firm staff, in-house counsel, and trained paralegal personnel. They are not directed at, marketed to, or intended for use by children under 13.

## 7. Confidentiality and professional privilege

Users — particularly advocates and law-firm staff — should note that case-folder contents passed to Claude Code may include material attracting attorney-client privilege, advocate-client confidentiality under Section 132 of the Bharatiya Sakshya Adhiniyam 2023 / Section 126 of the Indian Evidence Act 1872, or other professional-confidentiality obligations. Decisions about what material to supply to the plugin remain the user's professional responsibility and should be made consistent with applicable bar-council rules and client engagement terms.
