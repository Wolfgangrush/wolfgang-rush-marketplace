## 1. Data சேகரிப்பு

Wolfgang Rush plugin குடும்பம் **உங்கள் எந்த data-வையும் சேகரிக்காது, சேமிக்காது, அல்லது அனுப்பாது** — வெளியீட்டாளருக்கோ, எந்த மூன்றாம் தரப்பினருக்கோ.

## 2. Plugin-கள் எவ்வாறு இயங்குகின்றன

ஒவ்வொரு plugin-ம் **முற்றிலும் உள்ளூரில்** உங்கள் Claude Code installation-க்குள் இயங்குகிறது. Plugin-ன் agents, skills, மற்றும் templates உங்கள் உள்ளூர் filesystem-இலிருந்து படிக்கப்படுகின்றன. நீங்கள் வழங்கிய case-folder contents (facts, prior pleadings, statutory anchors, prayer briefs, exhibits) Claude Code-ஆல் உள்ளூரில் process செய்யப்படுகின்றன, மற்றும் சாதாரண Claude Code operation-ன் வழியில் உங்களுக்கும் Anthropic-ன் Claude API-க்கும் இடையே மட்டுமே அனுப்பப்படுகின்றன.

வெளியீட்டாளர் இந்த plugin-களுடன் தொடர்புடைய **எந்த** server, telemetry endpoint, analytics service, error-reporting service, license-verification service, அல்லது data-collection infrastructure-ஐயும் இயக்கவில்லை.

## 3. Anthropic-க்கு அனுப்பப்படும் Data

Plugin-ன் பயன்பாட்டிற்கான user prompts மற்றும் case-folder contents உங்கள் சொந்த Claude Code installation வழியாக Anthropic-ன் Claude API-க்கு அனுப்பப்படுகின்றன. அந்த data flow **Anthropic-ன் Privacy Policy** மூலம் நிர்வகிக்கப்படுகிறது, அது https://www.anthropic.com/legal/privacy-இல் கிடைக்கிறது.

வெளியீட்டாளர் அந்த data flow-ல் ஒரு தரப்பினர் அல்ல, user prompts அல்லது case-folder contents-க்கு எந்த அணுகலும் இல்லை, மற்றும் எந்த generated output-ன் நகலும் பெறவில்லை.

## 4. User account இல்லை

Plugin-களுக்கு எந்த user account, login, அல்லது அடையாள படிநிலையும் தேவையில்லை. எந்த registration, email signup, அல்லது licence-key activation-ம் இல்லை.
